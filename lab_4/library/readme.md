# 📚 Django Library Management System
 
A Django web application built to demonstrate core framework concepts as part of a Web Frameworks lab assignment.
 
---
 
## 🧩 Concepts Covered
 
- MVT Architecture (Model, View, Template)
- Routing & Middleware
- ORM & Database CRUD Operations
- User Authentication & Sessions
- Form Handling & Validation
- Custom Middleware (Logging, IP Blocking)
 
---
 
## ⚙️ Setup
 
```bash
git clone https://github.com/YOUR-USERNAME/django-library-demo.git
cd django-library-demo
python -m venv venv
source venv/bin/activate       # Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
